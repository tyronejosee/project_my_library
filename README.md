## âœ¨ Key Features

- **Immersive Hero Experience**: Dynamic hero sections with auto-playing carousels and smooth rotating text effects.
- **Comprehensive Library**: Dedicated sections for **Movies**, **Series**, and **Live TV**.
- **Deep Categorization**: Filter movies by genres (Action, Animation, Comedy, etc.) and browse specialized collections.
- **Advanced Search**: Fast, global search functionality with a sleek UI.
- **Interactive Dashboard**: Track your collection statistics and recently added content.
- **Micro-animations**: Subtle glassmorphism, noise patterns, and spring-based transitions for a premium feel.
- **Mobile Optimized**: Clean navigation and responsive layouts for all device sizes.
- **Data-Driven**: Uses structured JSON data for easy library updates and persistence.

## âš™ï¸ Installation

Clone the repository.

```bash
git clone git@github.com:tyronejosee/project_my_library.git
cd project_my_library
```

Install dependencies.

```bash
pnpm install
```

Set up environment variables, copy `.env.example` to `.env` and fill in the required values.

```bash
cp .env.example .env
```

Run the development server.

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
